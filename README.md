# TrufaBot

Bot de Discord modular en Node.js + TypeScript, enfocado en música y preparado para crecer con comandos extra.

## Estado
- Setup inicial hecho.
- Arquitectura documentada en `docs/discord-music-bot-architecture.md`.
- Guía de setup humano en `docs/user.md`.

## Ejecutar local
```bash
npm install
npm run dev
```

## Stack
- `discord.js`
- `@discordjs/voice`
- `distube`
- `typescript`
